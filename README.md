# Interfaz de Control E/S (Arduino - Processing)

Este trabajo práctico implementa el control y visualización bidireccional de Entradas (Pulsadores) y Salidas (LEDs) entre Arduino y PC, cumpliendo con los requisitos de la consigna.

##  Funcionalidades
* **Monitoreo:** Visualización en tiempo real del estado de los botones y LEDs en la interfaz.
* **Control:** Activación de salidas mediante clic en la pantalla o pulsadores físicos.
* **Comunicación:** Protocolo serial optimizado (sin `delay`) para multitarea.
